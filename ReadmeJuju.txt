Lancement du projet
*******************

- npm i : on réinstalle npm
- npm run start

Le projet
*********
Projet en binôme qui utilise l'API "Bêta-Series".
L'appel à l'API nous permet d'afficher l'ensemble des séries, et les détails de chacunes d'antre elles.

Langages
********
Front = React, Tailwind, HTML5
